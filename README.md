# SHSU Bingo

## Setup

### Yarn

Install yarn if you don't already have it installed: `npm i -g yarn`

### Install dependencies

Run `yarn`

## Scripts

### Development

Run `yarn dev` to start dev environment

### Production

Run `yarn build && yarn export && yarn start` to start the production server
